# [Click Your Trave](http://clickyourtravel.com.ve/) - [Freelancer](http://startbootstrap.com/template-overviews/freelancer/)

Needed Nodejs

## Getting Started
To install dependences

`npm install`

## Develop Mode
`gulp dev`

#Production
To compile all css, js and html run

`gulp default`
